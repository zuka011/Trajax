# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Faran, please report it privately by emailing ![faran.conduct@proton.me](mailto:faran.conduct@proton.me) with the subject line "Security Vulnerability - <Brief Description>" and a description of the issue, including steps to reproduce if possible. Please do not include any sensitive information in the email.

Please do not open a public GitLab issue for security vulnerabilities.

We will acknowledge receipt within a few days and aim to release a fix promptly.
